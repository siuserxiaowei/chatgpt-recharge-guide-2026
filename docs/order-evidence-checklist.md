# 订单和售后留痕清单

更新时间：2026-06-14

## 下单后必须保存

- 订单号。
- 查单密码。
- 接收邮箱。
- 商品名称。
- 付款时间。
- 付款金额。
- 支付截图或 USDT 交易哈希。
- 页面异常、账号状态或邮件延迟截图。

## 遇到异常先做什么

1. 先查订单状态，不要重复付款。
2. 检查邮箱收件箱和垃圾邮件。
3. 对照付款凭证确认金额和订单是否一致。
4. 准备订单号、查单密码、付款截图和异常截图。
5. 再联系售后核验。

## 不要提交

- 密码。
- 验证码。
- 长期有效登录凭证。
- 与订单无关的隐私信息。
